Major Projects

  • Yelp Photo Classification Task Prof.Dimitri Samaras Fall 2018

    Built a multi-instance multi-label classifier that would predict the labels describing the business attributes of restaurants from the user submitted pictures. We have built multiclass SVM as well as an ensemble model consisting of a support vector machine, logistic regression, and random forest using SIFT and pretrained CNNs like AlexNet and VGG, to extract features. The data is taken from Kaggle and consists of nearly 0.2 million pictures.

  • Action Recognition using LSTMs in Pytorch Fall 2018

    Built an LSTM model for predicting the action that is being done in the video. The features were extracted from a pretrained VGG network. The LSTM was compared with SVM to analyze its performance.

  • Predicting word analogies by building word embeddings using Word2Vec model Fall 2018

    Built a Noise Contrastive Estimation based model using Tensorflow and word embeddings to predict word analogies.

  • Human vs Cars prediction model using Computer Vision Winter 2016

    Built a SVM classifier and used histogram of oriented gradients (HOG) technique to extract features from objects and recognize them using OpenCV. Used images of different cars and human beings to train the classifier.

  • Tokenization techniques for biomedical information retrieval, Prof.Prasenjit Majumder Winter 2016

    Implemented various tokenizers to tokenize TREC-2014 data which contains 7,50,000 documents on biomedical discipline and used terrier to evaluate precision and recall based on 30 biomedical queries.

  • Built a backend module for a web application for M.G Coaching Centre. Winter 2016

    Implemented backend modules where parents can send notification to faculty and faculty can send notifications to parents regarding any announcement. Used Django for implementing these modules.

  • Estimation of correlation coefficient and regression line of given 2D Or 3D points using parallel programming (CUDA), Prof.Bhaskar Chaudary Autumn 2015

    Given any 2D/3D points as input, correlation coefficient and regression line can be used for statistical interpretations. The input can be related to population, stock markets etc. Used CUDA for parallelizing the code.

  • Implemented logistics system using socket programming, Prof.Laxminarayana Pillutla Autumn 2015

    Implemented a real-time replica of logistics systems, where there is a central supplier and relevant recipients request for goods..The whole project was implemented in C using socket programming.

  • Voice controlled wheelchair for the handicapped, Prof.Amit Bhatt Autumn 2015

    Made a prototype of a wheelchair that works with voice commands given by the patient. Made two android application so that even a caretaker can control the wheelchair with help of bluetooth communication. The whole project was implemented using arduino, android and electronics.

  • Tiny unix shell, Prof.Anish Mathuria Winter 2015

    Developed a tiny unix-like interactive tsh shell which supports signal processing, process management and job control executable functions as well as some in-built functions.

  • Robot controlled with mobile phones using DTMF, Prof.Sanjeev Gupta Winter 2014

    Built a robot which can be controlled with keys on a typical cellular phone which generates Dual Tone Multi Frequency (DTMF). For a particular key pressed on the mobile phone robot moves in a particular direction. Even robot has a mobile phone attached to it.
